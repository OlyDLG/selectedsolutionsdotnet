1.1.6) State the order and linearity or nonlinearity of \(\Frac{d^3y}{dx^3} + x\Frac{dy}{dx} + (\cos^2 x)y = x^3\).

Sln: The highest‑order derivative occurring in the equation is three, so this equation is \(\box{3^{\text{rd}}\text{ order}}.\)  It is \(\box{\text{linear}}\) because it is of the form \(a_0(x)y^{(n)} + a_1(x)y^{(n-1)} + ... + a_n(x)y = g(x)\).

1.1.14) Verify that \(y = e^{x^2}\DefInt{0}{x}\) \(e^{-t^2} dt + e^{x^2}\) is a solution of \(y‚Äô - 2xy = 1\).

Sln: \(y‚Äô = 2xe^{x^2}\DefInt{0}{x}\) \(e^{-t^2} dt + e^{x^2}(e^{-x^2}) + 2xe^{x^2} = 2xe^{x^2}\DefInt{0}{x}\) \(e^{-t^2} dt + 1 + 2xe^{x^2}\) so \(y‚Äô - 2xy = \color{red}{\cancel{2xe^{x^2}\DefInt{0}{x}\normalsize e^{-t^2} dt}} + 1 + \color{blue}{\cancel{2xe^{x^2}}} - \color{red}{\cancel{2xe^{x^2}\DefInt{0}{x}\normalsize e^{-t^2} dt}} - \color{blue}{\cancel{2xe^{x^2}}} = 1.~~~\blacksquare\)

1.1.18) Which values of \(r\) make \(y = e^{rx}\) a solution of \(y‚Äô‚Äô‚Äô - 3y‚Äô‚Äô + 2y‚Äô = 0\)?

Sln: First note that for \(y=f(x)\) to be a solution of a differential equation, it must make the equation a true statement for all \(x\) over which the DE is supposed to hold.  Now, \(y = e^{rx} \implies y‚Äô = re^{rx}, y‚Äô‚Äô = r^2e^{rx}, y‚Äô‚Äô‚Äô = r^3e^{rx}\) so \(0 = y‚Äô‚Äô‚Äô - 3y‚Äô‚Äô + 2y‚Äô = r^3e^{rx} - 3r^2e^{rx} + 2re^{rx} = (r^3 - 3r^2 + 2r)e^{rx} \implies r^3 - 3r^2 + 2r = 0\) because \(e^{rx} \ne 0\) for all \(x\) and \(r\).  \(0 = r^3 - 3r^2 + 2r = r(r-1)(r-2) \implies \box{r = 0, 1, 2}\) as the values of \(r\) which make \(y = e^{rx}\) a solution of \(y‚Äô‚Äô‚Äô - 3y‚Äô‚Äô + 2y‚Äô = 0\).  (The reader should check that \(y = e^{0x}=1, y = e^{1x} = e^x,\) and \(y = e^{2x}\) all do indeed make \(y‚Äô‚Äô‚Äô - 3y‚Äô‚Äô + 2y‚Äô = 0\) a true statement.)

1.1.20) Which values of \(r\) make \(y = x^r, x \gt 0\) a solution of \(x^2y‚Äô‚Äô - 4xy‚Äô + 4y = 0\)?

Sln: \(y = x^r, x \gt 0 \implies y‚Äô = rx^{r-1}, y‚Äô‚Äô = r(r-1)x^{r-2} \implies x^2y‚Äô‚Äô - 4xy‚Äô + 4y = r(r-1)x^{r-2+2} - 4rx^{r-1+1} + 4x^r = (r^2 - r - 4r + 4)x^r = 0\) \(\implies r^2 - 5r + 4 = 0\) since \(x^r \ne 0\) for all \(x \gt 0\).  Now, \(0 = r^2 - 5r + 4 = (r-1)(r-4) \implies \box{r = 1, 4}\) as the values of \(r\) that make \(x^r\) a solution of \(x^2y‚Äô‚Äô - 4xy‚Äô + 4y = 0\) for all \(x \gt 0\).
Check: \(y = x^1 \implies y‚Äô = 1, y‚Äô‚Äô = 0 \implies x^2y‚Äô‚Äô - 4xy‚Äô + 4y = x^2(0) - 4x(1) + 4x \overset{\chec}{=} 0\); \(y = x^4 \implies y‚Äô = 4x^3, y‚Äô‚Äô = 12x^2 \implies x^2y‚Äô‚Äô - 4xy‚Äô + 4y = x^2(12x^2) - 4x(4x^3) + 4(x^4) = (12 - 16 + 4)x^4 \overset{\chec}{=} 0\).

1.1.26) State the order and linearity or nonlinearity of \(u_t + uu_x = 1 + u_{xx}\).

Sln: (Note: this PDE is the one (spatial) dimension version of the conservation of momentum portion of the Navier-Stokes equations, which give a "field description" of the motion of (in this version) an incompressible, viscous fluid.)
The highest‑order derivative occurring in the equation is two (\(u_{xx}\)), so this is a \(\box{\text{second order}}\) equation. (However, for at least a couple of reasons you‚Äôll learn later, it is useful to distinguish the order of a PDE with respect to each of its independent variables, so we will learn to characterize this equation as second‑order in \(x\), first‑order in \(t\).)  It is \(\box{\text{nonlinear}}\) because of the presence of the \(uu_x\) term.

1.1.32) Given: \(f, g\) are twice‑differentiable functions; Verify: \(u(x,t) = f(x-at) + g(x+at)\) is a solution of \(a^2u_{xx} = u_{tt}\).

Sln: By the Chain Rule for multivariable functions, \( u_x = f‚Äô(x-at)\Frac{\partial}{\partial x}(x-at) + g‚Äô(x+at)\Frac{\partial}{\partial x}(x+at) = f‚Äô(x-at)(1) + g‚Äô(x+at)(1) = f‚Äô(x-at) + g‚Äô(x+at)\) and thus, similarly, \(u_{xx} = f‚Äô‚Äô(x-at) + g‚Äô‚Äô(x+at)\). Also: \( u_t = f‚Äô(x-at)\Frac{\partial}{\partial t}(x-at) + g‚Äô(x+at)\Frac{\partial}{\partial t}(x+at) = f‚Äô(x-at)(-a) + g‚Äô(x+at)(a) = -af‚Äô(x-at) + ag‚Äô(x+at) \implies\) \(u_{tt} = (-a)^2f‚Äô‚Äô(x-at) + a^2g‚Äô‚Äô(x+at) = a^2(f‚Äô‚Äô(x-at) + g‚Äô‚Äô(x+at)) = a^2u_{xx}\) as was to be shown.  (Note: this is the d‚ÄôAlembert solution of the one (spatial) dimension wave equation—the reader should read about what the form of this solution implies about this solution‚Äôs physical nature.)

2.1.4) \(y‚Äô + (1/x)y = 3\cos 2x,~~~~~x \gt 0\)

Sln: (I prefer to solve these using the method rather than the formula.)  This is of the form \(y‚Äô + p(x)y = g(x)\) with \(p(x) = 1/x, g(x) = 3\cos 2x\); thus the integrating factor \(\mu(x) = \exp\left(\Int\Frac1x dx\right) = e^{\log x} = x\).  Multiplying through by \(x\) (which we can do without concern since it‚Äôs stipulated that \(x \gt 0\)), we get:

\(xy‚Äô + y = \Frac d{dx}(xy)\) \(= 3x\cos 2x \implies xy = \Int3x\cos 2x dx = \Frac34\left(2x\sin 2x + \cos 2x\right) + C \implies\)

\[\box{y = \Frac32\sin 2x + \Frac{3\cos 2x}{4x} + \Frac Cx}\] Check: \( y‚Äô = \Frac32 (2)\cos 2x + \Frac{4x(-6\sin 2x) - 4(3\cos 2x)}{16x^2} - \Frac C{x^2} = 3\cos 2x - \Frac{3\sin 2x}{2x} - \Frac{3\cos 2x}{4x^2} - \Frac C{x^2}\),
\( (1/x)y = \Frac{3\sin 2x}{2x} + \Frac{3\cos 2x}{4x^2} + \Frac C{x^2}\) so
\( y‚Äô + (1/x) = 3\cos 2x \cancel{- \Frac{3\sin 2x}{2x} + \Frac{3\sin 2x}{2x}} \cancel{- \Frac{3\cos 2x}{4x^2} + \Frac{3\cos 2x}{4x^2}} \cancel{- \Frac C{x^2} + \Frac C{x^2}}.~~\chec\)

2.1.8) \((1+x^2)y‚Äô + 4xy = (1+x^2)^{-2}\)

Sln: Noting that \(1+x^2 \ne 0\) for all real \(x\), divide through by it to yield the equivalent DE \(y‚Äô + 4x(1+x^2)^{-1}y = (1+x^2)^{-3} \equiv\) \(y‚Äô + p(x)y = g(x)\) with \(p(x) = 4x(1+x^2)^{-1}, g(x) = (1+x^2)^{-3}\).  Thus the integrating factor \(\mu(x) = \exp\left(\Int\Frac{4x}{1+x^2}dx\right) = \exp\left(2\Int\Frac{du}{u}\right) \text{ (letting } u = 1+x^2 \implies du = 2xdx\text{) } = e^{2\log u} = (1+x^2)^2\).  Multiplying the \(y‚Äô + p(x)y = g(x)\) form of the equation through by \(\mu(x)\) yields:
\((1+x^2)^2 y‚Äô + 4x(1+x^2)y = \Frac d{dx}\left[(1+x^2)^2 y\right] = \Frac1{1+x^2} \implies (1+x^2)^2y = \int\Frac1{1+x^2}dx = \tan^{-1}x + C \implies\) \[\box{y = \frac{\tan^{-1}x + C}{(1+x^2)^2}}\] Check: \( (1+x^2)y‚Äô = (1+x^2)\left[\Frac{(1+x^2)^2(1+x^2)^{-1} - 4x(1+x^2)(\tan^{-1}x + C)}{(1+x^2)^4}\right] = (1+x^2)^{-2}(1 - 4x(\tan^{-1}x + C)\)
\(4xy = 4x(\tan^{-1}x + C)(1+x^2)^{-2}\) so \((1+x^2)y‚Äô + 4xy = (1+x^2)^{-2} \cancel{- 4x(\tan^{-1}x + C)(1+x^2)^{-2} + 4x(\tan^{-1}x + C)(1+x^2)^{-2}}~~\chec\)

Problems 2.1.12 and 2.1.16: find the solution of the given initial value problem (IVP).

2.1.12) \( y‚Äô + \Frac2x y = \Frac{\cos x}{x^2},~~~y(\pi) = 0,~~~x \gt 0\)

Sln: \(p(x) = \Frac2x \implies \mu(x) = \exp\left(\small{\int\frac2x dx}\right) = x^2\); multiplying through, we obtain \(x^2y‚Äô + 2xy = \Frac{d}{dx}\left(x^2y\right) = \cos x \implies x^2y = \sin x + C\); the IC requires \(\pi^2(0) = 0 = \sin\pi + C = C\) so the solution is \[\box{y = \frac{\sin x}{x^2}}\] IC check: \(\Frac{\sin\pi}{\pi^2} = 0~~\chec\)
DE check: \(y‚Äô = \Frac{x^2\cos x - 2x\sin x}{x^4} = \Frac{\cos x}{x^2} - \Frac{2\sin x}{x^3}\)
\(\Frac2x y = \Frac{2\sin x}{x^3} \implies y‚Äô + \Frac2x y = \Frac{\cos x}{x^2} \cancel{- \Frac{2\sin x}{x^3} + \Frac{2\sin x}{x^3}}~~\chec\)

2.1.16) \(xy‚Äô + (x+1)y = x,~~~y(\log 2) = 1\)

Sln: At \(x = 0\) the DE reduces to \(y(x) \equiv 0\), and there is no differentiable function consistent with this and the specified initial condition, so assume \(x \ne 0\).  Then the given DE is equivalent to \(y‚Äô + (1+\Frac1x)y = 1 \equiv y‚Äô + p(x)y = g(x)\) with \(g(x) = 1, p(x) = 1 + \Frac1x \implies \mu(x) = \exp\left[\int\left(1+\frac1x\right)dx\right] = e^{x+\log x} = xe^x\).  Multiplying this through the \(y‚Äô + p(x)y = g(x)\) form of the equation yields \(xe^x y‚Äô + (x+1)e^x y = \Frac{d}{dx}(xe^x y) = xe^x \implies\) \(xe^x y = \Int xe^x dx = (x-1)e^x + C\).  The IC requires \(\log 2 e^{\log 2}(1) = \cancel{2\log 2} = (\log 2 - 1)e^{\log 2} + C = \cancel{2\log 2} - 2 + C \implies C = 2\) so we have \(xe^x y = (x-1)e^x + 2 \implies\) \[\box{y = 1 - \frac{1 - 2e^{-x}}x}\] IC check: \(1 - \Frac{1-2e^{-\log 2}}{\log 2} = 1 - \Frac0{\log 2} = 1~~\chec\)
DE check: \(xy‚Äô = x\left(0 - \Frac{x(2e^{-x}) - (1)(1 - 2e^{-x})}{x^2}\right) = \Frac{1 - 2e^{-x} - 2xe^{-x}}{x}\)
\((x+1)y = x + \cancel{1 - 1} + 2e^{-x} - \Frac{1 - 2e^{-x}}{x} = x - \Frac{1 - 2e^{-x} - 2xe^{-x}}{x} \implies\)
\(xy‚Äô + (x+1)y = \cancel{\Frac{1 - 2e^{-x} - 2xe^{-x}}{x}} + x \cancel{- \Frac{1 - 2e^{-x} - 2xe^{-x}}{x}} = x~~\chec\)

2.1.21) Solve the IVP \(\Frac{dy}{dx} = \Frac1{e^y - x},~~~y(1)=0\); hint: consider \(x\) as a function of \(y\).

Sln: Following the hint, begin by rewriting the equation as \(\Frac{dx}{dy} = e^y - x \equiv x‚Äô + p(y)x = g(y)\) with \(g(y) = e^y, p(y) = 1 \implies \mu(y) = \exp\left(\Int dy \right) = e^y\); multiplying through gives \(e^{2y} = e^y x‚Äô + e^y x = \Frac{dx}{dy}(e^y x) \implies\) \(e^y x = \Int e^{2y}dy = \Frac12e^{2y} + C\); the IC gives \(e^0(1) = 1 = \Frac12e^{2(0)} + C \implies C = \Frac12\); thus \(e^{y} x = \Frac{e^{2y} + 1}2 \implies x = \Frac{e^y + e^{-y}}2 = \cosh y \implies \) \[\box{y = \cosh^{-1}x,~~x \ge 1}\] IC check: \(\cosh^{-1}(1) = 0~~\chec\)
DE check: \(\Frac{d}{dy}\cosh y = \sinh y = \Frac12(e^y - e^{-y}) = \Frac12(2e^y - (e^y + e^{-y}) = e^y - \cosh y = e^y - x \implies \Frac{dy}{dx} = \Frac1{e^y - x}~~\chec\)

2.1.22b) Show that \(\phi(x) = c/x\) is a solution of \(y‚Äô + y^2 = 0~(x \gt 0)\) if and only if \(c=0\) or \(c=1\).

Pf: \(\phi‚Äô(x) = -c/x^2 \implies \phi‚Äô + \phi^2 = -\Frac{c}{x^2} + \Frac{c^2}{x^2}\); setting this equal to zero and utilizing the given that \(x \gt 0\) to multiply through by \(x^2 \implies c(c-1) = 0 \implies c = 0\) or \(c=1\) as the only values of \(c\) which make the equation a true statement.\(~~~\blacksquare\)

2.1.24) Given \(y_1(x)\) is a solution of \(y‚Äô + p(x)y = 0\) and \(y_2(x)\) is a solution of \(y‚Äô + p(x)y = g(x)\), show that \(y_1 + y_2\) is also a solution of \(y‚Äô + p(x)y = g(x)\).

Pf: \((y_1 + y_2)‚Äô + p(x)(y_1 + y_2) = (y_1‚Äô + p(x)y_1) + (y_2‚Äô + p(x)y_2) = 0 + g(x)\) (because it is given that \(y_1‚Äô + p(x)y_1 = 0\) and \(y_2‚Äô + p(x)y_2 = g(x)\)) \( = g(x)\), i.e., \(y_1 + y_2\) is a solution of \(y‚Äô + p(x)y = g(x).~~~\blacksquare\)
