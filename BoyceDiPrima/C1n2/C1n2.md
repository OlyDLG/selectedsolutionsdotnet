### Chapter 1, Section 1

<a name="P1.1.6" class="goback" onclick="winhisback()">__1.1.6__)</a>
State the order and linearity or nonlinearity of $\Frac{d^3y}{dx^3} + x\Frac{dy}{dx} + (\cos^2 x)y = x^3$.

__Sln__: The highest&#8209;order derivative occurring in the equation is three, so this equation is $\boxed{3^{\text{rd}}\text{ order}}.$&nbsp; It is $\boxed{\text{linear}}$ because it is of the form $a_0(x)y^{(n)} + a_1(x)y^{(n-1)} + ... + a_n(x)y = g(x)$.
<br><br>

<a name="P1.1.14" class="goback" onclick="winhisback()">__1.1.14__)</a> 
Verify that $y = e^{x^2}\Int{0}{x}$ $e^{-t^2} dt + e^{x^2}$ is a solution of $y‚Äô - 2xy = 1$.

__Sln__: $y‚Äô = 2xe^{x^2}\Int{0}{x}$ $e^{-t^2} dt + e^{x^2}(e^{-x^2}) + 2xe^{x^2} = 2xe^{x^2}\Int{0}{x}$ $e^{-t^2} dt + 1 + 2xe^{x^2}$ so $y‚Äô - 2xy = \color{red}{\cancel{2xe^{x^2}\Int{0}{x}\normalsize e^{-t^2} dt}} + 1 + \color{blue}{\cancel{2xe^{x^2}}} - \color{red}{\cancel{2xe^{x^2}\Int{0}{x}\normalsize e^{-t^2} dt}} - \color{blue}{\cancel{2xe^{x^2}}} = 1.~~~\blacksquare$
<br><br>

<a name="P1.1.18" class="goback" onclick="winhisback()">__1.1.18__)</a>
Which values of $r$ make $y = e^{rx}$ a solution of $y‚Äô‚Äô‚Äô - 3y‚Äô‚Äô + 2y‚Äô = 0$?

__Sln__: First note that for $y=f(x)$ to be a solution of a differential equation, it must make the equation a true statement <i>for all</i> $x$ over which the DE is supposed to hold.&nbsp; Now, $y = e^{rx} \implies y‚Äô = re^{rx}, y‚Äô‚Äô = r^2e^{rx}, y‚Äô‚Äô‚Äô = r^3e^{rx}$ so $0 = y‚Äô‚Äô‚Äô - 3y‚Äô‚Äô + 2y‚Äô = r^3e^{rx} - 3r^2e^{rx} + 2re^{rx} = (r^3 - 3r^2 + 2r)e^{rx} \implies r^3 - 3r^2 + 2r = 0$ because $e^{rx} \ne 0$ for all $x$ and $r$.&nbsp; $0 = r^3 - 3r^2 + 2r = r(r-1)(r-2) \implies \boxed{r = 0, 1, 2}$ as the values of $r$ which make $y = e^{rx}$ a solution of $y‚Äô‚Äô‚Äô - 3y‚Äô‚Äô + 2y‚Äô = 0$.&nbsp; (The reader should check that $y = e^{0x}=1, y = e^{1x} = e^x,$ and $y = e^{2x}$ all do indeed make $y‚Äô‚Äô‚Äô - 3y‚Äô‚Äô + 2y‚Äô = 0$ a true statement.)
<br><br>

<a name="P1.1.20" class="goback" onclick="winhisback()">__1.1.20__)</a>
Which values of $r$ make $y = x^r, x \gt 0$ a solution of $x^2y‚Äô‚Äô - 4xy‚Äô + 4y = 0$?

__Sln__: $y = x^r, x \gt 0 \implies y‚Äô = rx^{r-1}, y‚Äô‚Äô = r(r-1)x^{r-2} \implies x^2y‚Äô‚Äô - 4xy‚Äô + 4y = r(r-1)x^{r-2+2} - 4rx^{r-1+1} + 4x^r = (r^2 - r - 4r + 4)x^r = 0$ $\implies r^2 - 5r + 4 = 0$ since $x^r \ne 0$ for all $x \gt 0$.&nbsp; Now, $0 = r^2 - 5r + 4 = (r-1)(r-4) \implies \boxed{r = 1, 4}$ as the values of $r$ that make $x^r$ a solution of $x^2y‚Äô‚Äô - 4xy‚Äô + 4y = 0$ for all $x \gt 0$.<br>
Check: $y = x^1 \implies y‚Äô = 1, y‚Äô‚Äô = 0 \implies x^2y‚Äô‚Äô - 4xy‚Äô + 4y = x^2(0) - 4x(1) + 4x \overset{\checkmark}{=} 0$; $y = x^4 \implies y‚Äô = 4x^3, y‚Äô‚Äô = 12x^2 \implies x^2y‚Äô‚Äô - 4xy‚Äô + 4y = x^2(12x^2) - 4x(4x^3) + 4(x^4) = (12 - 16 + 4)x^4 \overset{\checkmark}{=} 0$.
<br><br>

<a name="P1.1.26" class="goback" onclick="winhisback()">__1.1.26__)</a>
State the order and linearity or nonlinearity of $u_t + uu_x = 1 + u_{xx}$.

__Sln__: (Note: this PDE is the one (spatial) dimension version of the conservation of momentum portion of the [Navier-Stokes equations](https://en.wikipedia.org/wiki/Navier%E2%80%93Stokes_equations), which give a &quot;field description&quot; of the motion of (in this version) an incompressible, viscous fluid.)<br>
The highest&#8209;order derivative occurring in the equation is two ($u_{xx}$), so this is a $\boxed{\text{second order}}$ equation. (However, for at least a couple of reasons you‚Äôll learn later, it is useful to distinguish the order of a PDE with respect to each of its independent variables, so we will learn to characterize this equation as second&#8209;order in $x$, first&#8209;order in $t$.)&nbsp; It is $\boxed{\text{nonlinear}}$ because of the presence of the $uu_x$ term.
<br><br>

<a name="P1.1.32" class="goback" onclick="winhisback()">__1.1.32__)</a>
Given: $f, g$ are twice&#8209;differentiable functions; Verify: $u(x,t) = f(x-at) + g(x+at)$ is a solution of $a^2u_{xx} = u_{tt}$. 

__Sln__: By the [Chain Rule for multivariable functions](https://en.wikipedia.org/wiki/Chain_rule#Multivariable_case), $ u_x = f‚Äô(x-at)\Frac{\partial}{\partial x}(x-at) + g‚Äô(x+at)\Frac{\partial}{\partial x}(x+at) = f‚Äô(x-at)(1) + g‚Äô(x+at)(1) = f‚Äô(x-at) + g‚Äô(x+at)$ and thus, similarly, $u_{xx} = f‚Äô‚Äô(x-at) + g‚Äô‚Äô(x+at)$. Also: $ u_t = f‚Äô(x-at)\Frac{\partial}{\partial t}(x-at) + g‚Äô(x+at)\Frac{\partial}{\partial t}(x+at) = f‚Äô(x-at)(-a) + g‚Äô(x+at)(a) = -af‚Äô(x-at) + ag‚Äô(x+at) \implies$ $u_{tt} = (-a)^2f‚Äô‚Äô(x-at) + a^2g‚Äô‚Äô(x+at) = a^2(f‚Äô‚Äô(x-at) + g‚Äô‚Äô(x+at)) = a^2u_{xx}$ as was to be shown.&nbsp; (Note: this is the [d‚ÄôAlembert solution](https://en.wikipedia.org/wiki/Wave_equation#Algebraic_approach) of the one (spatial) dimension wave equation&mdash;the reader should read about what the form of this solution implies about this solution‚Äôs physical nature.)
<br><br>

### Chapter 2, Section 1

Problems 2.1.4 and 2.1.8: find the general solution of the given differential equation.

<a name="P2.1.4" class="goback" onclick="winhisback()">__2.1.4__)</a>
$y‚Äô + (1/x)y = 3\cos 2x,~~~~~x \gt 0$

__Sln__: (I prefer to solve these using the method rather than the formula.)&nbsp; This is of the form $y‚Äô + p(x)y = g(x)$ with $p(x) = 1/x, g(x) = 3\cos 2x$; thus the integrating factor $\mu(x) = \exp\left(\Int{}{}\!\!\small\Frac1x dx\right) = e^{\log x} = x$.&nbsp; Multiplying through by $x$ (which we can do without concern since it‚Äôs stipulated that $x \gt 0$), we get:<br>

$xy‚Äô + y = \Frac d{dx}(xy)$ $= 3x\cos 2x \implies xy = \Int{}{}$ $\!\!3x\cos 2x dx = \Frac34\left(2x\sin 2x + \cos 2x\right) + C \implies$ 

$$\boxed{y = \Frac32\sin 2x + \Frac{3\cos 2x}{4x} + \Frac Cx}$$
Check: $ y‚Äô = \Frac32 (2)\cos 2x + \Frac{4x(-6\sin 2x) - 4(3\cos 2x)}{16x^2} - \Frac C{x^2} = 3\cos 2x - \Frac{3\sin 2x}{2x} - \Frac{3\cos 2x}{4x^2} - \Frac C{x^2}$,<br>
$ (1/x)y = \Frac{3\sin 2x}{2x} + \Frac{3\cos 2x}{4x^2} + \Frac C{x^2}$ so<br>
$ y‚Äô + (1/x) = 3\cos 2x \cancel{- \Frac{3\sin 2x}{2x} + \Frac{3\sin 2x}{2x}} \cancel{- \Frac{3\cos 2x}{4x^2} + \Frac{3\cos 2x}{4x^2}} \cancel{- \Frac C{x^2} + \Frac C{x^2}}.~~\checkmark$
<br><br>

<a name="P2.1.8" class="goback" onclick="winhisback()">__2.1.8__)</a>
$(1+x^2)y‚Äô + 4xy = (1+x^2)^{-2}$

__Sln__: Noting that $1+x^2 \ne 0$ for all real $x$, divide through by it to yield the equivalent DE $y‚Äô + 4x(1+x^2)^{-1}y = (1+x^2)^{-3} \equiv$ $y‚Äô + p(x)y = g(x)$ with $p(x) = 4x(1+x^2)^{-1}, g(x) = (1+x^2)^{-3}$.&nbsp; Thus the integrating factor $\mu(x) =$ $\displaystyle \exp\left(\int\frac{4x}{1+x^2}dx\right) = \exp\left(2\int\frac{du}{u}\right) \text{ (letting } u = 1+x^2 \implies du = 2xdx\text{) } = e^{2\log u} = (1+x^2)^2$.&nbsp; Multiplying the $y‚Äô + p(x)y = g(x)$ form of the equation through by $\mu(x)$ yields:<br>
$(1+x^2)^2 y‚Äô + 4x(1+x^2)y = \Frac d{dx}\left[(1+x^2)^2 y\right] = \Frac1{1+x^2} \implies (1+x^2)^2y = \int\Frac1{1+x^2}dx = \tan^{-1}x + C \implies$
$$\boxed{y = \Frac{\tan^{-1}x + C}{(1+x^2)^2}}$$
Check: $ (1+x^2)y‚Äô = (1+x^2)\left[\Frac{(1+x^2)^2(1+x^2)^{-1} - 4x(1+x^2)(\tan^{-1}x + C)}{(1+x^2)^4}\right] = (1+x^2)^{-2}(1 - 4x(\tan^{-1}x + C)$<br>
$4xy = 4x(\tan^{-1}x + C)(1+x^2)^{-2}$ so $(1+x^2)y‚Äô + 4xy = (1+x^2)^{-2} \cancel{- 4x(\tan^{-1}x + C)(1+x^2)^{-2} + 4x(\tan^{-1}x + C)(1+x^2)^{-2}}~~\checkmark$
<br><br>

Problems 2.1.12 and 2.1.16: find the solution of the given initial value problem (IVP).

<a name="P2.1.12" class="goback" onclick="winhisback()">__2.1.12__)</a>
$ y‚Äô + \Frac2x y = \Frac{\cos x}{x^2},~~~y(\pi) = 0,~~~x \gt 0$

__Sln__: $p(x) = \Frac2x \implies $ $\mu(x) = \exp\left(\Int{}{}\right.$ $\!\!\left.\Frac2x dx\right) = x^2$; multiplying through, we obtain $x^2y‚Äô + 2xy = \Frac{d}{dx}\left(x^2y\right) = \cos x \implies x^2y = \sin x + C$; the IC requires $\pi^2(0) = 0 = \sin\pi + C = C$ so the solution is
$$\boxed{y = \frac{\sin x}{x^2}}$$
IC check: $\Frac{\sin\pi}{\pi^2} = 0~~\checkmark$<br>
DE check: $y‚Äô = \Frac{x^2\cos x - 2x\sin x}{x^4} = \Frac{\cos x}{x^2} - \Frac{2\sin x}{x^3}$<br>
$\Frac2x y = \Frac{2\sin x}{x^3} \implies y‚Äô + \Frac2x y = \Frac{\cos x}{x^2} \cancel{- \Frac{2\sin x}{x^3} + \Frac{2\sin x}{x^3}}~~\checkmark$
<br><br>

<a name="P2.1.16" class="goback" onclick="winhisback()">__2.1.16__)</a>
$xy‚Äô + (x+1)y = x,~~~y(\log 2) = 1$

__Sln__: At $x = 0$ the DE reduces to $y(x) \equiv 0$, and there is no differentiable function consistent with this and the specified initial condition, so assume $x \ne 0$.&nbsp; Then the given DE is equivalent to $y‚Äô + (1+\Frac1x)y = 1 \equiv y‚Äô + p(x)y = g(x)$ with $g(x) = 1, p(x) = 1 + \Frac1x \implies$ $\mu(x) = \displaystyle\exp\left[\int\left(1+\frac1x\right)dx\right] = e^{x+\log x} = xe^x$.&nbsp; Multiplying this through the $y‚Äô + p(x)y = g(x)$ form of the equation yields $xe^x y‚Äô + (x+1)e^x y = \Frac{d}{dx}(xe^x y) = xe^x \implies$ $xe^x y = \Int{}{}$ $\!\! xe^x dx = (x-1)e^x + C$.&nbsp; The IC requires $\log 2 e^{\log 2}(1) = \cancel{2\log 2} = (\log 2 - 1)e^{\log 2} + C = \cancel{2\log 2} - 2 + C \implies C = 2$ so we have $xe^x y = (x-1)e^x + 2 \implies$
$$\boxed{y = 1 - \frac{1 - 2e^{-x}}x}$$
IC check: $1 - \Frac{1-2e^{-\log 2}}{\log 2} = 1 - \Frac0{\log 2} = 1~~\checkmark$<br>
DE check: $xy‚Äô = x\left(0 - \Frac{x(2e^{-x}) - (1)(1 - 2e^{-x})}{x^2}\right) = \Frac{1 - 2e^{-x} - 2xe^{-x}}{x}$<br>
$(x+1)y = x + \cancel{1 - 1} + 2e^{-x} - \Frac{1 - 2e^{-x}}{x} = x - \Frac{1 - 2e^{-x} - 2xe^{-x}}{x} \implies$<br>
$xy‚Äô + (x+1)y = \cancel{\Frac{1 - 2e^{-x} - 2xe^{-x}}{x}} + x \cancel{- \Frac{1 - 2e^{-x} - 2xe^{-x}}{x}} = x~~\checkmark$
<br><br>

<a name="P2.1.21" class="goback" onclick="winhisback()">__2.1.21__)</a>
Solve the IVP $\Frac{dy}{dx} = \Frac1{e^y - x},~~~y(1)=0$; hint: consider $x$ as a function of $y$.

__Sln__: Following the hint, begin by rewriting the equation as $\Frac{dx}{dy} = e^y - x \equiv x‚Äô + p(y)x = g(y)$ with $g(y) = e^y, p(y) = 1 \implies \mu(y) = \exp\left(\Int{}{}\!\! dy \right) = e^y$; multiplying through gives $e^{2y} = e^y x‚Äô + e^y x = \Frac{dx}{dy}(e^y x) \implies$ $e^y x = \Int{}{}\!\! e^{2y}dy = \Frac12e^{2y} + C$; the IC gives $e^0(1) = 1 = \Frac12e^{2(0)} + C \implies C = \Frac12$; thus $e^{y} x = \Frac{e^{2y} + 1}2 \implies x = \Frac{e^y + e^{-y}}2 = \cosh y \implies $
$$\boxed{y = \cosh^{-1}x,~~x \ge 1}$$
IC check: $\cosh^{-1}(1) = 0~~\checkmark$<br>
DE check: $\Frac{d}{dy}\cosh y = \sinh y = \Frac12(e^y - e^{-y}) = \Frac12(2e^y - (e^y + e^{-y}) = e^y - \cosh y = e^y - x \implies \Frac{dy}{dx} = \Frac1{e^y - x}~~\checkmark$

<a name="P2.1.22b" class="goback" onclick="winhisback()">__2.1.22b__)</a>
Show that $\phi(x) = c/x$ is a solution of $y‚Äô + y^2 = 0~(x \gt 0)$ if and only if $c=0$ or $c=1$.
 
__Pf__: $\phi‚Äô(x) = -c/x^2 \implies \phi‚Äô + \phi^2 = -\Frac{c}{x^2} + \Frac{c^2}{x^2}$; setting this equal to zero and utilizing the given that $x \gt 0$ to multiply through by $x^2 \implies c(c-1) = 0 \implies c = 0$ or $c=1$ as the only values of $c$ which make the equation a true statement.$~~~\blacksquare$

__Pf__: $(y_1 + y_2)‚Äô + p(x)(y_1 + y_2) = (y_1‚Äô + p(x)y_1) + (y_2‚Äô + p(x)y_2) = 0 + g(x)$ (because it is given that $y_1‚Äô + p(x)y_1 = 0$ and $y_2‚Äô + p(x)y_2 = g(x)$) $ = g(x)$, i.e., $y_1 + y_2$ is a solution of $y‚Äô + p(x)y = g(x).~~~\blacksquare$
