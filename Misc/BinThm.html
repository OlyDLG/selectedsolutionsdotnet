Binomial Expansion Formula

\[(1+b)^N = \sum_{n=0}^N \left(\! \begin{array}{c} N \\ n \end{array} \!\right) b^{N-n}\] (Taking \(0!\) and \(0^0 \equiv 1.\))
Proof: By induction: it's true for \(N=0: \sum_{n=0}^0 \left(\! \begin{array}{c} N \\ n \end{array} \!\right)b^{N-n} = \left(\! \begin{array}{c} 0 \\ 0 \end{array} \!\right)b^{0-0} = 1 = (1+b)^0~\forall b\)

Now assume true for \(N \in \mathbb{W}\); we'll show this implies true for \(N+1\):

\[\sum_{n=0}^{N+1} \left(\! \begin{array}{c} N + 1\\ n \end{array} \!\right)b^{N-n+1} = \left(\! \begin{array}{c} N + 1\\ 0 \end{array} \!\right) b^{N+1} + \sum_{n=1}^{N} \left(\! \begin{array}{c} N + 1\\ n \end{array} \!\right)b^{N-n+1} + \left(\! \begin{array}{c} N + 1\\ N + 1 \end{array} \!\right) b^0 =\]
\[b^{N+1} + \sum_{n=1}^{N} \frac{(N+1)!}{n!(N+1-n)!} b^{N+1-n} + 1 = b^{N+1} + \sum_{n=1}^{N} \frac{(N - n + 1 + n)N!}{n!(N+1-n)!} b^{N+1-n} + 1 =\]
\[b^{N+1} + \sum_{n=1}^{N} \left[\frac{(N-n+1)N!}{n!(N-n+1)!} + \frac{nN!}{n!(N-(n-1))!}\right] b^{N+1-n} + 1 =\]
\[b^{N+1} + \sum_{n=1}^{N}\left(\! \begin{array}{c} N \\ n \end{array} \!\right) b^{N-n+1} + \sum_{n=1}^{N}\left(\! \begin{array}{c} N \\ n-1 \end{array} \!\right) b^{N-(n-1)} + 1 =\]
\[b\left[b^N + \sum_{n=1}^{N}\left(\! \begin{array}{c} N \\ n \end{array} \!\right) b^{N-n}\right] + \left[\sum_{n=0}^{N-1}\left(\! \begin{array}{c} N \\ n \end{array} \!\right) b^{N-n} + 1\right] = b \sum_{n=0}^{N}\left(\! \begin{array}{c} N \\ n \end{array} \!\right) b^{N-n} + \sum_{n=0}^{N}\left(\! \begin{array}{c} N \\ n \end{array} \!\right) b^{N-n} =\]
\[(b+1)(b+1)^N~(\because \sum_{n=0}^{N}\left(\! \begin{array}{c} N \\ n \end{array} \!\right) b^{N-n}) = (1+b)^N \text{by assumption}) =(1+b)^{N+1} \\\] \(\blacksquare\)

Corollary 1

\[(a+b)^N = \sum_{n=0}^{N}\left(\! \begin{array}{c} N \\ n \end{array} \!\right) a^n b^{N-n} \]

Proof: if \(a = 0\), it's true by (careful) inspection (and our definition \(0^0 \equiv 1\)), so assume \(a \ne 0.~\) Then \[(a + b)^N = a^N (1+(b/a))^N = a^N \sum_{n=0}^{N}\left(\! \begin{array}{c} N \\ n \end{array} \!\right) 1^n \left(\frac{b}{a}\right)^{N-n} = \sum_{n=0}^{N}\left(\! \begin{array}{c} N \\ n \end{array} \!\right) a^{N-(N-n)} b^{N-n} = \sum_{n=0}^{N}\left(\! \begin{array}{c} N \\ n \end{array} \!\right) a^n b^{N-n} \\\] \(\blacksquare\)

Note: this is why one proves the B.E.F. as above, and then derives the general result as a corollary; alternatively, if one insists on calling our Corollary 1 the B.E.F., one can call our B.E.F. a Lemma (but the Reader should recognize that it's really six one way, half-a-dozen the other.)

Corollary 2

\[\sum_{n=0}^{N}\left(\! \begin{array}{c} N \\ n \end{array} \!\right) = 2^N\]
Proof: \[\sum_{n=0}^{N}\left(\! \begin{array}{c} N \\ n \end{array} \!\right) = \sum_{n=0}^{N}\left(\! \begin{array}{c} N \\ n \end{array} \!\right) 1^n 1^{N-n} = (1+1)^N = 2^N\\\] \(\blacksquare\)

Note: this explains why the sum of each row of Pascal's Triangle is 2, raised to the power of the-row-number-minus-one (or simply the row number, if one starts numbering from zero).

(You'll probably never have to use these next six—except perhaps in an exam of some sort or another—they're just given to show some of the creative ways one can use this little tidbit of mathematics in combination with Euler's Formula.)

Corollary 3

\[\sum_{n=0}^N \left(\! \begin{array}{c} N \\ n \end{array} \!\right) \cos{[(2n-N)\theta]} = (2\cos\theta)^N\] \[\sum_{n=0}^N \left(\! \begin{array}{c} N \\ n \end{array} \!\right) \sin{[(2n-N)\theta]} = 0\] Proof: For \(\theta \in \mathbb{R}\), \[(2\cos \theta)^N = \left(e^{i\theta} + e^{-i\theta}\right)^N = \sum_{n=0}^N \left(\! \begin{array}{c} N \\ n \end{array} \!\right) e^{in\theta}e^{-i(N-n)\theta} = \sum_{n=0}^N \left(\! \begin{array}{c} N \\ n \end{array} \!\right) e^{i(2n-N)\theta} =\] \[\sum_{n=0}^N \left(\! \begin{array}{c} N \\ n \end{array} \!\right) \cos{[(2n-N)\theta]} + i\sin{[(2n-N)\theta]} \implies \text{both our results.}\\\] \(\blacksquare\)

Corollary 4

\[\Re{[(2i\sin{\theta})^N]} = \sum_{n=0}^N \left(\! \begin{array}{c} N \\ n \end{array} \!\right)(-1)^{N-n}\cos{[(2n-N)\theta]}\] \[\Im{[(2i\sin{\theta})^N]} = \sum_{n=0}^N \left(\! \begin{array}{c} N \\ n \end{array} \!\right)(-1)^{N-n}\sin{[(2n-N)\theta]}\] (where \(\Re\) and \(\Im\) are, respectively, the real and imaginary parts of their argument.)

\(\displaystyle\frac{d^n}{dx^n}(\sin^mx)\): Left to the Reader. 

Answer: \(\boxed{\frac{i^{m+n}}{2^m}\sum_{k=0}^m(-1)^k\left(\begin{array}{c} m \\ k \end{array}\right)(2k-m)^n\left(e^{i(2k-m)x}\right)}\)

\(\begin{eqnarray} \displaystyle\frac{d^{-n}}{dx^{-n}}(\sin^mx) & = & \frac{d^{-n}}{dx^{-n}}\left[\left(\frac{e^{ix}-e^{-ix}}{2i}\right)^m\right] = \frac1{(2i)^m}\frac{d^{-n}}{dx^{-n}} \sum_{k=0}^m \left(\begin{array}{c} m \\ k \end{array}\right) (-1)^{m-k} e^{ikx}e^{-i(m-k)x} = \frac1{(2i)^m}\sum_{k=0}^m \left(\begin{array}{c} m \\ k \end{array}\right) (-1)^{m-k} \frac{d^{-n}}{dx^{-n}} e^{i(2k-m)x} \\ & = & \displaystyle \left(\frac{-1}{2i}\right)^m\sum_{k=0}^m \left(\frac1{-1}\right)^k \left(\begin{array}{c} m \\ k \end{array}\right) i^{-n}(2k-m)^{-n} e^{i(2k-m)x} = \boxed{\frac{i^{m-n}}{2^m} \sum_{k=0}^m \left(\begin{array}{c} m \\ k \end{array}\right) \frac{(-1)^k}{(2k-m)^n} e^{i(2k-m)x}} \end{eqnarray}\)

where \(\displaystyle\frac{d^{-n}}{dx^{-n}} \equiv \underbrace{\int \int \ldots \int}_{n\text{ times}} (dx)^n\)
